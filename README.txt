An implemetation that uses geotools quickstart [1], to analyse GPS data.

Delaunay Triangulation
- Along with this tool, Java implementation of S-hull algorithm [4] for finding the Delaunay triangulation of a point set is available.

[1] https://www.geotools.org/
[2] http://docs.geotools.org/latest/userguide/tutorial/quickstart/index.html
[3] http://docs.geotools.org/latest/userguide/tutorial/quickstart/maven.html
[4] https://arxiv.org/abs/1604.01428
